15 Puzzle

My first ASCII game.

This is my first ASCII videogame and I programmed it in C. The player's goal is to sort the numbers from 1 to 15 by moving the tiles to the empty position. When the player has ordered the numbers from 1 to 15 the game will end and the new score will be saved in the record list. The player can use the w, s, a and d keys to move the numbers in the game matrix. I programmed the game controls using two functions included in conio.h. The first one is kbhit(), which allows me to check when a key is pressed, and the second one is getch(), which returns the ASCII value of the key pressed. I also made the game smoother by using the gotoxy(x,y) function insted of system("cls"). The gotoxy(x,y) function allows you to move the cursor of the terminal freely and therefore you can delete and print only the data that is modified and not the whole terminal. For saving the score in the ranking I used the linked lists. When I save a new score I read the Record.txt file and I create a list with all the scores. I insert the new score in the list by comparing it with the others in order to understand in which position it will be placed. In the end, I rewrite the txt file by reading the updated list.

Nyanvaders

My first Android game.

This is my first videogame for Android devices and I developed it using Game Maker Studio v1.4.9999. This game is a modernized version of Space Invaders and thanks to it I learned how to move objects and how to handle collisions between them. The aim of the player is to kill the enemies by dodging their bullets and grabbing bonus items. The level is infinite and I randomized the bonus items and the enemies in order to create an increasing difficulty in an infinite level. The player will keep playing until his life bar reaches 0, at which point will be game over. At the end of the game, if the player has reached a sufficiently high score, the new record will be saved by replacing the previous one.

Dungeon Shooter

A top-down shooter with randomly generated levels.

This is my second videogame for Android devices and I developed it using Game Maker Studio v1.4.9999. For this videogame I wrote an algorithm that allows the random generation of levels. The level map is divided into grass, the area where the player can move, and rocks, obstacles that block the player. The algorithm starts by drawing a tile of grass and then it will move one tile in a random direction choosing between up, down, right or left. Then it will draw a new grass tile in the new area and move again choosing between the four directions. It will continue this process a certain numbero of times and then it will fill the empty spaces of the map with rocks. At this point the player will start the game and the enemies and the bonus items will be randomly generated. The player have to kill all the enemies in the map in order to open the door that will allow him to access the next level.
